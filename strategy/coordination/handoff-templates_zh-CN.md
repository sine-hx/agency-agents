# 📋 NEXUS 交接模板

> NEXUS 管道中每种智能体间交接的标准化模板。一致交接防止上下文丢失 — 多智能体协调失败的第一原因。

---

## 1. 标准交接模板

用于任何智能体间工作转移。

```markdown
# NEXUS 交接文档

## 元数据
| 字段 | 值 |
|------|-----|
| **来自** | [智能体名称]（[部门]) |
| **到** | [智能体名称]（[部门]) |
| **阶段** | Phase [N] — [阶段名称] |
| **任务参考** | [Sprint Prioritizer backlog 任务 ID] |
| **优先级** | [关键 / 高 / 中 / 低] |
| **时间戳** | [YYYY-MM-DDTHH:MM:SSZ] |

## 上下文
**项目**：[项目名称]
**当前状态**：[已完成什么 — 具体说明]
**相关文件**：
- [file/path/1] — [包含什么]
- [file/path/2] — [包含什么]
**依赖**：[此工作依赖什么完成]
**约束**：[技术、时间线或资源约束]

## 交付请求
**需要什么**：[具体、可测量的交付物描述]
**验收标准**：
- [ ] [标准 1 — 可测量]
- [ ] [标准 2 — 可测量]
- [ ] [标准 3 — 可测量]
**参考资料**：[规格、设计、之前工作链接]

## 质量期望
**必须通过**：[此交付物具体质量标准]
**需要证据**：[完成证明是什么样的]
**交接下一步**：[谁接收输出，需要什么格式]
```

## 2. QA 反馈循环 — PASS

当 Evidence Collector 或其他 QA 智能体批准任务时使用。

```markdown
# NEXUS QA 结论：PASS ✅

## 任务
| 字段 | 值 |
|------|-----|
| **任务 ID** | [ID] |
| **任务描述** | [描述] |
| **开发者智能体** | [智能体名称] |
| **QA 智能体** | [智能体名称] |
| **尝试次数** | [N] / 3 |
| **时间戳** | [YYYY-MM-DDTHH:MM:SSZ] |

## 结论：PASS

## 证据
**截图**：
- 桌面 (1920x1080)：[filename/path]
- 平板 (768x1024)：[filename/path]
- 移动 (375x667)：[filename/path]

**功能验证**：
- [x] [验收标准 1] — 已验证
- [x] [验收标准 2] — 已验证
- [x] [验收标准 3] — 已验证

**品牌一致性**：已验证 — 颜色、排版、间距匹配设计系统
**无障碍**：已验证 — 键盘导航、对比度、语义 HTML
**性能**：[测量加载时间] — 在可接受范围内

## 备注
[任何观察、未来改进建议或正面亮点]

## 下一步行动
→ Agents Orchestrator：标记任务完成，推进 backlog 下一个任务
```

## 3. QA 反馈循环 — FAIL

当 Evidence Collector 或其他 QA 智能体拒绝任务时使用。

```markdown
# NEXUS QA 结论：FAIL ❌

## 任务
| 字段 | 值 |
|------|-----|
| **任务 ID** | [ID] |
| **任务描述** | [描述] |
| **开发者智能体** | [智能体名称] |
| **QA 智能体** | [智能体名称] |
| **尝试次数** | [N] / 3 |
| **时间戳** | [YYYY-MM-DDTHH:MM:SSZ] |

## 结论：FAIL

## 发现的问题

### 问题 1：[类别] — [严重程度：关键/高/中/低]
**描述**：[问题确切描述]
**期望**：[根据验收标准应发生什么]
**实际**：[实际发生什么]
**证据**：[截图文件名或测试输出]
**修复说明**：[解决的具体、可操作说明]
**需修改文件**：[确切文件路径]

### 问题 2：[类别] — [严重程度]
**描述**：[...]
**期望**：[...]
**实际**：[...]
**证据**：[...]
**修复说明**：[...]
**需修改文件**：[...]

[继续所有发现的问题]

## 验收标准状态
- [x] [标准 1] — 通过
- [ ] [标准 2] — FAILED（见问题 1)
- [ ] [标准 3] — FAILED（见问题 2)

## 重试说明
**给开发者智能体**：
1. 只修复上述列出的问题
2. 不引入新功能或变更
3. 所有问题解决后重新提交 QA
4. 这是尝试 [N]，最多 3 次

**如第 3 次失败**：任务将升级到 Agents Orchestrator
```

## 4. 升级报告

当任务超过 3 次重试时使用。

```markdown
# NEXUS 升级报告 🚨

## 任务
| 字段 | 值 |
|------|-----|
| **任务 ID** | [ID] |
| **任务描述** | [描述] |
| **开发者智能体** | [智能体名称] |
| **QA 智能体** | [智能体名称] |
| **尝试耗尽** | 3/3 |
| **升级到** | [Agents Orchestrator / Studio Producer] |
| **时间戳** | [YYYY-MM-DDTHH:MM:SSZ] |

## 失败历史

### 尝试 1
- **发现问题**：[摘要]
- **应用修复**：[开发者改了什么]
- **结果**：FAIL — [为什么仍失败]

### 尝试 2
- **发现问题**：[摘要]
- **应用修复**：[开发者改了什么]
- **结果**：FAIL — [为什么仍失败]

### 尝试 3
- **发现问题**：[摘要]
- **应用修复**：[开发者改了什么]
- **结果**：FAIL — [为什么仍失败]

## 根因分析
**为何任务持续失败**：[底层问题分析]
**系统性问题**：[这是一次性还是模式？]
**复杂度评估**：[任务是否恰当范围？]

## 推荐解决方案
- [ ] **重新分配**给不同开发者智能体（[推荐智能体])
- [ ] **分解**为更小子任务（[建议拆分])
- [ ] **修订方法** — 需架构/设计变更
- [ ] **接受**当前状态附文档限制
- [ ] **推迟**到未来迭代

## 影响评估
**阻塞**：[什么其他任务被此阻塞]
**时间线影响**：[如何影响整体日程]
**质量影响**：[如接受当前状态存在什么质量妥协]

## 需要决策
**决策者**：[Agents Orchestrator / Studio Producer]
**截止**：[何时需决策以避免进一步延误]
```

## 5. 阶段关卡交接

```markdown

## 过渡

| **关卡结果** | [PASSED / FAILED] |
| **时间戳** | [YYYY-MM-DDTHH:MM:SSZ] |

## 关卡标准结果
| # | 标准 | 阈值 | 结果 | 证据 |
|---|------|------|------|------|
| 1 | [标准] | [阈值] | ✅ PASS / ❌ FAIL | [证据参考] |
| 2 | [标准] | [阈值] | ✅ PASS / ❌ FAIL | [证据参考] |
| 3 | [标准] | [阈值] | ✅ PASS / ❌ FAIL | [证据参考] |

## 向前传递文档
```

## 7. 事件交接

在事件响应期间使用。

```markdown
# NEXUS 事件交接

## 事件
| 字段 | 值 |
|------|-----|
| **严重程度** | [P0 / P1 / P2 / P3] |
| **检测者** | [智能体或系统] |
| **检测时间** | [时间戳] |
| **分配给** | [智能体名称] |
| **状态** | [调查 / 缓解 / 解决 / 事后] |

## 描述
**发生什么**：[事件清晰描述]
**影响**：[谁/什么受影响及严重程度]
**时间线**：
- [HH:MM] — [事件]
- [HH:MM] — [事件]
- [HH:MM] — [事件]

## 当前状态
**受影响系统**：[列表]
**可用变通方案**：[是/否 — 如是描述]
**估计解决时间**：[时间估计]

## 已采取行动
1. [已采取行动及结果]
2. [已采取行动及结果]

## 交接上下文
**给下一响应者**：
- [已尝试什么]
- [尚未尝试什么]
- [疑似根因]
- [相关日志/指标需检查]

## 干系人沟通
**最后更新发送**：[时间戳]
**下次更新到期**：[时间戳]
**沟通渠道**：[更新发布位置]
```

## 使用指南

| 场景 | 使用模板 |
|------|----------|
| 分配工作给另一智能体 | 标准交接（#1） |
| QA 批准任务 | QA PASS（#2） |
| QA 拒绝任务 | QA FAIL（#3） |
| 任务超过 3 次重试 | 升级报告（#4） |
| 阶段间移动 | 阶段关卡交接（#5） |
| 迭代结束 | 迭代交接（#6） |
| 系统事件 | 事件交接（#7） |
