# 示例

本目录包含示例输出，展示代理团队的智能体如何协同应对真实任务。

## 为什么存在此目录

agency-agents 仓库定义了跨越工程、设计、营销、产品、支持、空间计算和项目管理领域的数十个专业智能体。但智能体定义本身并不能展示当你**一次性部署它们**执行单一任务时会发生什么。

这些示例回答了这个问题：*"当完整代理团队协作时实际看起来是什么样的？*

## 内容

### [nexus-spatial-discovery.md](./nexus-spatial-discovery.md)

**内容：** 一个完整的产品探索练习，8 个智能体并行工作评估软件机会并产出统一计划。

**场景：** Web 研究识别到 AI 智能体编排与空间计算交叉领域的机会。随后整个代理团队同时部署产出：

- 市场验证和竞争分析
- 技术架构（8 服务系统设计和完整 SQL 模式）
- 品牌策略和视觉识别
- 上市和增长计划
- 客户支持运营蓝图
- UX 研究计划含用户画像和旅程地图
- 35 周项目执行计划含 65 个迭代任务
- 空间界面架构规格

**使用的智能体：**
| 智能体 | 角色 |
|--------|------|
| Product Trend Researcher | 市场验证、竞争格局 |
| Backend Architect | 系统架构、数据模型、API 设计 |
| Brand Guardian | 定位、视觉识别、命名 |
| Growth Hacker | GTM 策略、定价、发布计划 |
| Support Responder | 支持层级、新用户引导、社区 |
| UX Researcher | 用户画像、旅程地图、设计原则 |
| Project Shepherd | 阶段计划、迭代、风险登记 |
| XR Interface Architect | 空间 UI 规格 |

**关键洞察：** 所有 8 个智能体并行运行，产出相互引用的一致计划，无需协调开销。输出展示了代理团队在单次会话中从"寻找机会"到"提供完整蓝图"的能力。
